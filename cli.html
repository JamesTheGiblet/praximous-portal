Get Started

The Sovereignty CLI

$ npm install -g dso-cli
$ cd my-project
$ dso generate readme --ai
# ✓ README.md generated and updated.
$ dso identity create --name "My Web3 ID"
# ✓ Decentralized identity created and secured.
$ dso deploy --service my-dapp
$ # ✓ Service deployed to decentralized network.

Control From Your Terminal

The Sovereignty CLI (dso-cli) brings the power of digital self-ownership directly into your development workflow.

  • Manage decentralized identities and credentials.
  • Automate documentation with the AI README tool.
  • Deploy applications to decentralized hosting platforms.
  • Script and automate complex sovereignty workflows.
